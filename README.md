# Algorithmic Automative Adaptive Acreage
_This repository contains code for STEM project that plants Pseudocereals in a hot environment by providing a cool environment for the plants. The project uses a combination of algorithms, automation, and adaptive technology to optimize the growth of the plants in a challenging environment in addition to AGENTIC AI integeration. It also supports a web interface for monitoring and controlling the system._
## Hardware Requirements
- ESP32 S3 microcontroller
- Temperature and humidity sensors
- 2 Fans (one for cooling system (radiator) and one for ventilation)
- Power supply 12V
- Step down converter (12V to 5V)
- 4-6 Mosfets
- 1 or 2 Relays
- 12V water pump (cooling system)
- 5V water pump (irrigation)
- 220 ohm resistors
- 10k ohm resistors (pull down resistors)
- Jumper wires
- Breadboard
- Diodes
- Capacitors
- Light intensity sensor
